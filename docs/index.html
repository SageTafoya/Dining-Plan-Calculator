Dining Dollars Calculator






  1. Enter the date of the end of the term OSU (Fall = Dec 13)
  2. Log into Starez and select Meal Plan and dining
  3. Copy your current dining plan balance with "CTRL+c"
  4. Return to this page and paste "CTRL+v" balance into "current term ballance"
  5. Dollarsigns "$" commas ",", and periods "." are ok for ballance
